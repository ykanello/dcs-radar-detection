How it behaves
 Case 1: Enemy pops up briefly
  RED EWR detects airborne BLUE player
  Flag 11 gets set
  Detection disappears before 180 sec
  No escalation
  Result: SAMs / alert state only
 Case 2: Enemy stays in the picture
  RED EWR detects airborne BLUE player
  Flag 11 gets set
  Detection continues long enough
  Flag 12 gets set
  Result: launch QRA / CAP / extra response
 Case 3: Radar group dies
  Script stops polling entirely
